# GPAssistant

This simple demo uses `llama-cpp-python` to run the Lingshu-7B model locally.

Download the Lingshu-7B GGUF model from [mradermacher/Lingshu-7B-GGUF](https://huggingface.co/mradermacher/Lingshu-7B-GGUF).

`MODEL_PATH` in `app.py` must point to the downloaded `.gguf` file before starting the application.
