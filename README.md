# Medical Multimodal Assistant

This project provides a simple Gradio chat interface for the [Lingshu-7B](https://huggingface.co/lingshu-medical-mllm/Lingshu-7B) medical model. The application allows you to upload images or a short video and ask questions about them. Replies are generated by the model on GPU when available.

## Features

- Multimodal input with images or a single short video
- Streaming text generation
- Optional system prompt and token limit controls

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the application:
   ```bash
   python app.py
   ```
3. Open the printed local URL in your browser.

You can adjust the maximum number of images processed from a video by setting `MAX_NUM_IMAGES` in your environment.
