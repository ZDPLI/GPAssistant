# Medical Multimodal Assistant

This project provides a simple Gradio chat interface for the [Lingshu-7B](https://huggingface.co/lingshu-medical-mllm/Lingshu-7B) medical model. The application allows you to upload images or a short video and ask questions about them. Replies are generated by the model on GPU when available.

## Features

- Multimodal input with images or a single short video
- Streaming text generation
- Optional system prompt and token limit controls

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. (Optional) Pre-download the model or point to an existing local copy to avoid
   Hugging Face timeouts:
   ```bash
   huggingface-cli download lingshu-medical-mllm/Lingshu-7B --local-dir ./model --local-dir-use-symlinks False
   export MODEL_PATH="$(pwd)/model"
   export MODEL_CACHE_DIR="$MODEL_PATH"  # optional cache location
   export HF_HUB_OFFLINE=1
   ```
   If `MODEL_PATH` is set to a directory with the model files, the application
   will load them directly from disk in offline mode. Optionally set
   `MODEL_CACHE_DIR` to reuse that directory as the Transformers cache. You can
   still override the checkpoint with `MODEL_ID` if needed.
3. Run the application:
   ```bash
   python app.py
   ```
4. Open the printed local URL in your browser.

You can adjust the maximum number of images processed from a video by setting `MAX_NUM_IMAGES` in your environment.
